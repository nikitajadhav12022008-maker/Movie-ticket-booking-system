##🎬 Movie Ticket Booking System (Python CLI):
A console-based movie ticket booking application written in Python.
The program allows users to book cinema tickets with real-world conditions such as age restrictions, membership discounts, weekend pricing, and seat categories.
---

##🚀 Features:
- Age-based booking validation
- Evening show restrictions
- Membership discount system
- Weekend extra charges
- Multiple seat categories (Premium, Gold, Regular)
- Continuous booking option
- Input validation handling
- Clean and modular code using functions
---

##🧠 Pricing Rules:
-Condition| Effect
-Age < 18| Cannot book ticket
-Evening show| Only 21+ or members allowed
-Member & Age ≥ 21| ₹3 discount
-Weekend or Evening| ₹2 extra charge
-Premium seat| ₹5 service charge
-Gold seat| ₹3 service charge
-Regular seat| ₹1 service charge
---
